About Application

“Glvrd Style Checker” helps to find problems in Russian-language texts: fillers, journalese, and corporate jargon.

The application uses the external service called “Главред” to perform the analysis. More details about the service can be found on the service creators' site (in Russian).

The application works with the Microsoft Word 2013 and later, Word 2016 for Mac, and Word Online.

Privacy Policy

The application does not collect or store any user private information.

The text selected for analysis got sent to the external “Главред” (in Russian) service for analysis.

The “Главред” service does not collect or store users’ private information. The sent text is used only for analysis and is not persisted on the server. The service stores anonymous statistics of check rules found in texts. The “Главред” service uses Yandex Metrika and Google Analytics analytics services to track usage of the service. A detailed description of the «Главред» service is available on the service’s site (in Russian).

How to Use

Select text to start checking. The application will perform the analysis, display the selected text in a pane, and highlight the suspicious phrases. Click the highlighted phrase in a pane to view a comment and fixing suggestions.
